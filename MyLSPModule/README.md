✨ 功能特性
🔒 隐藏指定联系人​
支持将指定好友从会话列表、联系人列表中隐藏，保护隐私不被轻易发现。
🚫 屏蔽朋友圈​
禁止指定联系人的朋友圈动态展示，不显示其朋友圈入口及内容。
📞 屏蔽语音通话​
拦截来自指定联系人的语音通话请求，避免被打扰或暴露关系。
📹 屏蔽视频通话​
拦截指定联系人的视频通话请求，防止误接或强制挂断。
👥 隐藏通讯录​
对指定联系人进行通讯录级隐藏，使其在联系人列表中不可见。
🔍 隐藏主页搜索​
禁止指定联系人通过微信首页搜索框被检索到。
💬 隐藏聊天记录搜索​
在聊天记录全局搜索中屏蔽指定联系人的历史消息

📲 模块入口
打开 微信 → 我 → 设置 → 更多 → 关怀模式​
👉 长按「开启」​ 进入密友管理界面
💬 使用方式
在 聊天页面输入命令：
#add
即可将当前联系人加入密友列表
